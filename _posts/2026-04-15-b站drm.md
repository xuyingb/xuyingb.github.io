## 准备工作

### 下载所需工具

- [Firefox 浏览器](https://www.firefox.com/zh-CN/download/all/desktop-developer/win64/zh-CN/)
- [Widevine L3 Decrypter](https://addons.mozilla.org/zh-CN/firefox/addon/widevine-l3-decrypter/)
- [BBDown](https://github.com/nilaoda/BBDown/releases)
- [mp4decrypt.exe](https://www.bento4.com/downloads/)
 

## 测试步骤

### 1. 下载音视频流（不合并）

```bash
BBDown.exe https://www.bilibili.com/cheese/play/ep1302284 --skip-mux
```

### 2. 获取解密密钥

参考下图获取 DRM 解密所需的 Key：

![获取key示例](./MtjvpdPF3bWoQ2H0Rfz6lK.webp)

### 3. DRM 解密

使用 `mp4decrypt.exe` 分别解密音频和视频：

```bash
# 解密音频
mp4decrypt.exe --key d8f66b93db284984b4e7fc50d71278ff:73ac4286e5a74f078abcbf3f8f40a6f0 "voice.m4a" "voice_.m4a"

# 解密视频
mp4decrypt.exe --key d8f66b93db284984b4e7fc50d71278ff:73ac4286e5a74f078abcbf3f8f40a6f0 "video.mp4" "video_.mp4"
```

> **注意**：`--key` 参数格式为 `密钥ID:解密密钥`

### 4. 合并音视频

使用 `ffmpeg` 合并解密后的文件：

```bash
ffmpeg.exe -i voice_.m4a -i video_.mp4 -c:v copy -c:a copy output.mp4
```
