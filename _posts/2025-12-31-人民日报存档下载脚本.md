## 📦 存档资源推荐

### 1. 数据库（推荐，仅限在线查看）

> 如果以下方法不可用，可尝试使用本地图书馆的数据库（如湖北省图书馆提供《党报党刊资料库》，免费办理电子读者证即可访问）。

**以湖北省图书馆为例：**

1. 访问 [https://www.library.hb.cn/](https://www.library.hb.cn/)，点击登录  
2. 使用支付宝扫描二维码，办理电子读者证  
3. 设置登录密码  
4. 登录后进入 [数字资源页](https://www.library.hb.cn/resource/szzy/)，找到 **人民日报数据库**，选择“馆外访问”

### 2. 在线网站（可下载）

- [https://reader.jojokanbao.cn/support/](https://reader.jojokanbao.cn/support/)（支持下载）  
- [https://cn.govopendata.com/renminribao/](https://cn.govopendata.com/renminribao/)

### 3. 网盘

- [OneDrive 存档](https://onedrive.live.com/?redeem=aHR0cHM6Ly8xZHJ2Lm1zL3UvcyFBajJKQzFoQlRscXpoOEFFOTY0SEQ4TWM2SEF6alE%5FZT1RTzhibWY&id=B35A4E41580B893D%21122884&cid=B35A4E41580B893D)

### 4. PT（Private Tracker）

- 站点如 **btschool（学校）**、**m-team（馒头）** 包含 1946–2025 年几乎全部日期的存档  
- 可请有账号的朋友帮忙下载  
- 其他 PT 站点也可能存在，自行查找即可

## 🛠 脚本说明

从官网 [https://paper.people.com.cn/rmrb/](https://paper.people.com.cn/rmrb/) 下载 PDF 并合并。

### 运行环境

- 需要本地安装 **Node.js**（建议高版本）

### 使用方法

```bash
npm i
node ./run.js
```
